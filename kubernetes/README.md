# Autonomous Release Orchestration Platform

## Project Overview

A complete DevOps project demonstrating CI/CD, Infrastructure as Code, Configuration Management, Monitoring, and Containerization.

## Technologies Used

* AWS EC2
* Git & GitHub
* Docker
* Jenkins
* Prometheus
* Grafana
* Terraform
* Ansible
* Kubernetes

## CI/CD Flow

GitHub → Jenkins → Docker Build → Docker Deployment

## Monitoring Flow

Node Exporter → Prometheus → Grafana

## Infrastructure as Code

Terraform used for EC2 and Security Group provisioning.

## Configuration Management

Ansible used for package installation and server automation.

## Kubernetes

Deployment and Service YAML files created for Flask application deployment.
